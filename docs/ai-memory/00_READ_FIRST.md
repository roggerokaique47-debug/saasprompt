# AI Memory System — Índice e Como Usar

Bem-vindo ao "Sistema Operacional do Projeto" do NovaFlow AI.
Se você é uma Inteligência Artificial (ChatGPT, Claude, Gemini, Cursor, Windsurf, Cline etc) ou um novo Engenheiro entrando neste projeto, **esta pasta é a sua única fonte da verdade de arquitetura**.

**O código é a verdade de execução. Os documentos são a verdade de arquitetura. Os dois nunca podem divergir.**
